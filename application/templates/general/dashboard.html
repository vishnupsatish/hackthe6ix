About Dyna

Dyna allows you to chat with your friends while creating your own custom commands that are executable over the cloud! Put your code to the test and test out various commands including finding the weather, performing internet searches and so many more!

Friends

Invite your friends to Dyna today, and you can use your commands all while chatting away! Simply share ask your friend for their username, add them in the Chat bar and that's it! All in a few clicks.

Basics

Start by going over to the commands tab and adding your first command. Use any of our multiple supported languages including C++, Python and Java. You can start with a command of any difficulty, even with a simple Hello, World! To set up a command, set a trigger phrase that will start the execution of your command when it is sent. Then add your code that will be executed as your command runs. Your code can be as simple as a print statement or it can even incorporate hard concepts like AI and Machine Learning!

Once you have your command set up, test it out by going into a chat with our bot or with a friend. You can then type in your keyword and watch your command be executed!

Finally, have fun setting more commands of different difficulties! Send any feedback or suggestions to hackthe6ix.project@gmail.com and we'll be glad to hear them!
